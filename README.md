# Dev & Design at OneIMS

## Installing

```bash
# From Source
git clone https://github.com/bokhariLovesYou/devndesignatoneims.git
cd devatoneims
```

then install dependencies

```bash
  npm install
```

## Extra options

```bash
# Run a production build, locally
gatsby build

# Serve a production build, locally
gatsby serve
```
